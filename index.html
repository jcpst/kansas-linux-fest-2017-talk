Using Arch Linux as your primary operating system

Principles

  • simple
  • modern
  • pragmatic
  • user centric
  • versatile

simple

  • upstream packages stay vanilla
  • downstream changes are minimal; limited to things like configuration for distro specific system file paths
  • configuration is in the hands of the user

modern

  • rolling release
  • install once, continuously upgrade
  • latest everything

pragmatic

  • functionality over ideology
  • both oss AND proprietary software packages are available

user centric

  • user-centric over user-friendly
  • fulfill the needs of contributors
  • community-driven

versatile

  • a general-purpose distribution
  • user builds system as needed
  • powerful packaging system
